How It Works

1. Register

Create your donor profile with blood group, phone number, city, and optional live location.

2. Search Nearby

Detect your location and search blood donors sorted by nearest available distance.

3. Send Requests

Send normal or emergency requests with simple status tracking and duplicate prevention.

4. Manage Responses

Accept or reject incoming requests and monitor sent requests in real time.

Important Note

This demo stores data in your browser using localStorage. It is good for showcasing the project, but a real production system should use a database and proper authentication.
